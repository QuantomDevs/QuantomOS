# Notes Widget

Create and edit markdown-formatted notes directly on your dashboard.

## Overview

The Notes Widget provides a simple note-taking interface with markdown support. Create quick notes, to-do lists, reminders, or documentation snippets right on your dashboard.

## Use Cases

- Quick notes and reminders
- To-do lists
- Server documentation
- Maintenance logs
- Command snippets
- IP address lists

## Features

- **Markdown Support**: Full markdown formatting
- **Syntax Highlighting**: For code blocks
- **Auto-save**: Changes save automatically
- **Preview Mode**: Toggle between edit and preview
- **Resizable**: Adjust size based on content

## Markdown Support

Supported markdown features:
- **Headers**: `# H1`, `## H2`, `### H3`
- **Bold**: `**text**` or `__text__`
- **Italic**: `*text*` or `_text_`
- **Lists**: Ordered (1. ) and unordered (- or *)
- **Links**: `[text](url)`
- **Code**: Inline `code` and code blocks
- **Checkboxes**: `- [ ]` and `- [x]`
- **Tables**: Markdown tables
- **Quotes**: `> quote`

## Setup Instructions

1. Add "Notes" widget
2. Click to edit
3. Write content in markdown
4. Changes save automatically
5. Toggle preview mode to see formatted output

## Tips

1. **Size**: Resize based on content length
2. **Multiple Notes**: Add separate widgets for different topics
3. **Code Blocks**: Use triple backticks for code
4. **Organization**: Use headers to structure notes

## Examples

### Quick To-Do List
```markdown
# Server Tasks
- [ ] Update Plex
- [x] Backup configs
- [ ] Check disk space
```

### Server Info
```markdown
## Server IPs
- Plex: 192.168.1.100
- NAS: 192.168.1.101
- Router: 192.168.1.1
```

### Code Snippet
```markdown
### Docker Compose
\`\`\`yaml
services:
  app:
    image: example
\`\`\`
```

## Troubleshooting

**Changes Not Saving**: Check browser console and network connectivity

**Markdown Not Rendering**: Verify markdown syntax is correct

**Widget Too Small**: Resize widget to accommodate content
