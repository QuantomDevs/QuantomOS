# Date & Time Widget

Display current date, time, and timezone information.

## Overview

The Date & Time Widget provides a real-time clock and calendar display. It automatically updates every minute and supports multiple time zones and date formats.

## Use Cases

- Quick time reference on your dashboard
- Monitor time in different time zones
- Display date and day of the week
- Track time for scheduling and time management

## Configuration Options

| Setting | Description | Default |
|---------|-------------|---------|
| **Time Format** | 12-hour or 24-hour | 12-hour |
| **Show Seconds** | Display seconds | No |
| **Show Date** | Display current date | Yes |
| **Show Day** | Display day of the week | Yes |
| **Time Zone** | Custom timezone | Auto-detect |
| **Date Format** | Format for date display | MM/DD/YYYY |

## Setup Instructions

1. Click the `+` button and select "Date & Time"
2. Configure display preferences
3. Choose timezone (or use auto-detect)
4. Select date format preference
5. Save and position on dashboard

## Features

- **Real-time Updates**: Updates every minute (or every second if enabled)
- **Timezone Support**: Display time in any timezone
- **Flexible Formatting**: Customize date and time display
- **Auto-timezone Detection**: Automatically uses your local timezone

## Tips

- Minimum size: 2x1 grid cells
- Add multiple widgets for different timezones
- Pair with Calendar widget for complete scheduling solution
- Disable seconds for cleaner look and less CPU usage

## Troubleshooting

**Time Not Updating**: Refresh the page or check browser console

**Wrong Timezone**: Edit widget and select correct timezone

**Date Format Issues**: Change format in widget settings
